Rules

Hash initials by name. Same person, same color, every surface. Predictability beats personalization.

Two letters max. "DC" for Dani Castillo; "D" for one-word names. Never three.

Project tile = palette story. Pick one of mesh / peach-coral / orchid / dark-magenta and stick to it for that project's life.

Placeholders are diagonal stripes. Never grey rectangles, never broken-image icons. Always labeled with what goes there.

Don't mix mesh and pastels in the same lockup. Choose either the warm spectrum or the feature pastel — not both.

When in doubt, ask. A real photo from the user beats every generated stand-in. Leave the slot empty and prompt for an upload.
